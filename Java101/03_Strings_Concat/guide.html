About

Strings, which are widely used in Java programming, they are a sequence of characters. In the Java programming language, strings are objects.The Java platform provides the String class to create and manipulate strings.

String class has utility functions to perform different kind of operations on strings. Some of the useful functions are explained below

concat():
The String concat() method concatenates the specified string to the end of current string. Another method to concatenate two string is using concatenation operator (+)
