A variable is a named location that stores a value. Values are things that can be printed, stored and (as weâ€™ll see later) operated on.
To store a value, you have to create a variable. Since the values we want to store are strings, we declare that the new variable is a string: 
String studentName;

This statement is a declaration, because it declares that the variable named studentName has the type String. Each variable has a type that determines what kind of values it can store. For example, the int type can store integers, and the String type can store strings

Assignment
Now that we know how to declare a variable , we want to store values. We do that with an assignment statement 
String studentName = "John"; // give studentName the value "studentName"
int marks = 80; // assign the value 80 to marks
char grade = 'A'; // set grade to 'A'
  • When you declare a variable, you create a named storage location.
  • When you make an assignment to a variable, you give it a value
