A variable sentence is defined with James likes Maths class. Change the value of sentence to James likes English class 
public class Snippet {		
		public static void main(String args[]) {
			String  sentence = "James likes Maths class";

			//Change the order to "James likes English class"
			sentence  = sentence.replace("Maths", "English");
			System.out.println(sentence);

			//assign length with the length of the new order string.
			int length = sentence.length();
			System.out.println("Length of the variable 'sentence' is: "+length);
		}
	}
