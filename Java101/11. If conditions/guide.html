  2. Check if 'x' is positive by using chained conditionals
  3. check if 'x' is positive by using nested conditionals

Chained conditionals:
Sometimes you want to check for a number of related conditions and choose one of several actions. One way to do this is by chaining a series of if's and elses
These chains can be as long as you want, although they can be difficult to read if they get out of hand 

        if(){
            //Statements
        } else if() {
            //Statements
        } else{
             //Statements
        }
Nested conditionals:
In addition to chaining, you can also nest one conditional within another. 
        if(){
            //Statements
        } else {
            if() {
            //Statements
            } else{
                //Statements
            }
        }
